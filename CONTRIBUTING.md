# 🤝 Contributing to Pinkus YT-DLP GUI

First off, thank you for considering contributing to **Pinkus YT-DLP GUI**! It's people like you that make open-source tools awesome.

---

## 🕒 How Can I Contribute?

### 1. Reporting Bugs
- Use the **Red "🔴 Errors" Button** in the app to identify the issue.
- Check the [**Troubleshooting Guide**](TROUBLESHOOTING.md) to see if it's a known problem.
- Open an issue on GitHub with details about your OS, Python version, and the URL that failed.

### 2. Suggesting Enhancements
- If you have an idea for a new feature (like cloud upload, or a specific site support), feel free to open a feature request.

### 3. Pull Requests
- Fork the repository.
- Create a feature branch (`git checkout -b feature/AmazingFeature`).
- Commit your changes.
- Push to the branch.
- Open a Pull Request.

---

## 🛠️ Style Guidelines

- **Python**: Follow [PEP 8](https://www.python.org/dev/peps/pep-0008/).
- **UI**: We use `CustomTkinter`. Keep the dark-blue theme and rounded designs consistent.
- **Documentation**: Update the [**User Guide**](USER_GUIDE.md) if you change any UI features.

---

## ⚖️ Code of Conduct
Please be respectful and helpful in all interactions. We aim to keep this community technical, friendly, and empowering for all users.
